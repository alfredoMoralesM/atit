## Overview

This project is a full redesign of the ATIT website, developed as an academic team effort between **January 2024** and **August 2024**. It features a modern, responsive interface covering all key sections of the academy's digital presence.

---

## Sections

| Section | Description |
|---|---|
| **Home** | Hero carousel with academy highlights |
| **Programs** | Academic offerings and degrees |
| **Staff** | Faculty and administrative personnel |
| **Internships** | Residency program information |
| **Projects** | Student and research projects |
| **Videos** | Custom video streaming module |
| **Indicators** | Performance and academic metrics |
| **Interaction** | Contact and community features |
| **Gallery** | Photo gallery of events and activities |
| **Help** | Support and FAQs |

---

## Highlights

- 🎨 **Responsive Web Design (RWD)** across all pages, optimized for desktop and mobile
- 📹 **Custom video streaming module** built from scratch
- 🧑‍💻 **Frontend team lead** — managed sprint planning, code reviews, and Git workflows, resulting in ~30% improvement in delivery speed
- 🔀 **Standardized branching strategy and pull request workflow** to reduce merge conflicts and scale collaboration
